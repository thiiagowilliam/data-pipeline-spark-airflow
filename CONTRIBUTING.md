# Contributing

Contributions are welcome! We are glad that you are interested in contributing to this project.

## How to Contribute

1.  **Fork the repository**: Click the "Fork" button at the top right of the page.
2.  **Clone your fork**:
    ```bash
    git clone https://github.com/YOUR_USERNAME/data-pipeline-spark-airflow.git
    ```
3.  **Create a new branch**:
    ```bash
    git checkout -b feature/your-feature-name
    ```
4.  **Make your changes**: Implement your feature or bug fix.
5.  **Commit your changes**:
    ```bash
    git commit -m "feat: add your commit message"
    ```
6.  **Push to your fork**:
    ```bash
    git push origin feature/your-feature-name
    ```
7.  **Open a Pull Request**: Go to the original repository and open a pull request.

## Development Setup

Please refer to the `README.md` for instructions on how to set up the development environment.

## Code Style

Please follow the existing code style. We use `black` for code formatting and `flake8` for linting.

## Issues

If you find a bug or have a feature request, please open an issue on the GitHub repository.
